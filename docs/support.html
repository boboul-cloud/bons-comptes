Questions fréquentes

Comment créer une campagne ?

Ouvrez l'app, appuyez sur le bouton « + » en haut à droite, donnez un nom à votre campagne, ajoutez les participants et c'est parti.

Comment partager une campagne avec mes amis ?

Ouvrez votre campagne, appuyez sur « Partager ». Vous pouvez envoyer un lien web par SMS, e-mail ou tout autre moyen. Vos amis n'ont pas besoin d'installer l'app pour consulter les dépenses.

Mes données sont-elles en sécurité ?

Oui. Toutes vos données sont stockées uniquement sur votre appareil. Aucun serveur, aucun compte, aucune collecte de données. Le partage se fait via des liens contenant les données encodées, sans serveur intermédiaire.

Que contient la version Premium ?

La version Premium débloque les campagnes illimitées, l'export PDF, le scanner de tickets, le QR code SEPA, le partage de proximité, les sauvegardes et la Live Activity. C'est un achat unique, sans abonnement.

J'ai acheté Premium mais mes fonctionnalités ne sont pas débloquées

Allez dans Réglages et appuyez sur « Restaurer les achats ». Si le problème persiste, contactez-nous par e-mail avec votre identifiant Apple.

Comment supprimer mes données ?

Supprimez vos campagnes dans l'app ou désinstallez l'application. Toutes les données seront effacées car rien n'est stocké sur un serveur.

L'app fonctionne-t-elle hors ligne ?

Oui, entièrement. L'app ne nécessite aucune connexion internet. Vous pouvez créer des campagnes, ajouter des dépenses et consulter les soldes sans réseau.

Comment fonctionne le partage de proximité ?

Le partage de proximité utilise Bluetooth et Wi-Fi pour envoyer une campagne directement aux appareils à proximité, sans internet. Les deux appareils doivent avoir l'app installée.

Guides utiles

  • Répartition personnalisée : lors de l'ajout d'une dépense, choisissez « Montants personnalisés » ou « Pourcentages » pour une répartition sur mesure.
  • QR Code SEPA : dans l'écran des soldes, appuyez sur un remboursement suggéré pour générer un QR code bancaire que le débiteur peut scanner avec son app bancaire.
  • Scanner de tickets : appuyez sur l'icône scanner dans une campagne, prenez une photo du ticket et les articles seront détectés automatiquement.
  • Sauvegardes : dans les réglages, créez des sauvegardes manuelles de vos données et restaurez-les si nécessaire.

Configuration requise

iOS 17.0 ou version ultérieure. Compatible iPhone et iPad.
